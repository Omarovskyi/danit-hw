1.Завантаження Linux
   BIOS/UEFI(первинна ініціалізація обладнання, основні компоненти системи)->Bootloader(Вибір та завантаження ядра операційної системи зазвичай Grub)->Kernel Loading(завантаження самого ядра)
   ->Init(перший процес системи)-> User login(авторизація користувача)-> Desktop Environment(запуск робочого стола або домашня директорія користувача)-> System services (запуск системних      сервісів і користувацьких додатків).

2.Як переглянути системні журнали?
   команда : journalctl

3.-rw------- : Опишіть цей дозвіл. Як додати прапорець виконуваного файлу?
 
  -rw------- : для користувача owner надані права read,write/ для Group та Other закриті права на цей файл

 Як додати прапорець виконуваного файлу?
   командою chmod +x імя файлу(якщо для всіх або chmod (можна обрати U,G,O)+x імя файлу)

4. Яка різниця між apt і dpkg?
  
  apt - працює з пакетами 
    
  dpkg - працює з файлами пакетів
